## Master

- Added `enforce-min-coverage` rule.

```js
"flowtype-errors/enforce-min-coverage": [2, 50]
```

- Added `stopOnExit` setting.

```js
"settings": {
  "flowtype-errors": {
    "stopOnExit": "true"
  }
},
```
